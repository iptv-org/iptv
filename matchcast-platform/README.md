# 📡 MatchCast IPTV Platform

MatchCast is a lightweight IPTV-style streaming platform built from playlist aggregation concepts.

## 🚀 Features
- Express API backend
- Playlist proxy endpoint
- Docker support
- Extensible architecture for live sports streams

## 📁 Structure
- server/ → API backend
- docker-compose.yml → deployment setup

## ▶️ Run Locally
```bash
cd server
npm install
npm start
```

## 🐳 Run with Docker
```bash
docker-compose up --build
```
